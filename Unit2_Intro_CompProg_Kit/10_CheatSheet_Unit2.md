Unit 2 — Quick Cheat Sheet
Prepared: 2025-09-02

- Program: ordered instructions producing output from input.
- Assembler → assembly to machine. Compiler → source to machine/bytecode. Interpreter → executes source directly.
- Compiler phases: Lexical → Syntax → Semantic → IR → Optimize → CodeGen → Link.
- Errors: Syntax (compile), Runtime (execute), Logical (wrong output).
- Testing: Unit, Integration, System, Acceptance, Regression.
- Test design: Use equivalence partitioning & boundary value analysis for robust test suites.
